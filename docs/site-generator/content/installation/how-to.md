## Instalation
* ### from pypi
```
pip install django-zarinpal
```

* ### from github
```
git submodule add https://github.com/glyphack/djano-zarinpal
git submodule init
git submodule upadte
```
it will creates a folder in your project you can import zarinpal from django-zarinpal/zarinpal
